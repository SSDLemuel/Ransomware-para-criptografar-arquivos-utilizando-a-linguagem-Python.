# Ransomware-para-criptografar-arquivos-utilizando-a-linguagem-Python.
Fazendo implementação de um ransomware utilizadno linguagem Python, consistem em 3 arquivos:
- encrypter (arquivo para criptografar mensagem)
- decrypter (arquivo para descriptografar mensagem)
- teste (arquivo alvo que será manipulado)

# O Resultado obtido será um arquivo criptografado com nome:
- teste.txt.ransomwaretroll
